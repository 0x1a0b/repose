== What HTTP methods does *Repose* support?
Only the following HTTP methods are supported by *Repose*.

- `GET`
- `PUT`
- `POST`
- `DELETE`
- `HEAD`
- `OPTIONS`
- `PATCH`
- `TRACE`

Requests with any other HTTP method will be rejected with a response code of `400`.
